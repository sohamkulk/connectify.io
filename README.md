# Connectify.io

A real-time full-stack chat application built with the MERN stack and Socket.io.

## Features

- User Authentication (Signup / Login / Logout)
- Profile Image Upload
- Real-time Messaging with Socket.io
- Online / Offline Status
- Edit Messages (double click)
- Delete Messages for Everyone (double click)
- Typing Indicator
- Search Users
- Indian Date & Time on Messages (IST)
- User Profile Modal on Click
- Responsive Design (Mobile + Desktop)
- JWT Authentication with HTTP-only Cookies

## Tech Stack

**Frontend:**
- React.js (Vite)
- Tailwind CSS
- Socket.io Client
- Axios
- Lucide React (Icons)
- React Router DOM

**Backend:**
- Node.js
- Express.js
- MongoDB (Mongoose)
- Socket.io
- JWT (JSON Web Tokens)
- Bcrypt.js
- Multer (File Upload)
- Cookie Parser
